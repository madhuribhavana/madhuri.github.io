NUTANIX, INVISIBLE INFRASTRUCTURE

Redesigning the virtual machine creation flow

üìú  CONTEXT
Nutanix is a cloud computing software company that provides hyconverged infrastructure and virtualisation software such as Prism, Calm, Xi etc. While Prism is Nutanix's flagship product, enabling end-to-end VM and storage management and global network visualization, one of the core flow which is the Virtual Machine Creation flow currently does not support all user roles, all customizations and is not compatible with on-cloud infrastructure. The goal of the project is to design a flow that addresses these issues. As a side project, I also conceptualized a VM chatbot that can deploy VMs for end-users.
